# Food Manager

Food Manager is a full stack recipe management web application where users can add view edit and delete their favorite recipes All recipe data is connected with a MySQL database so every change is stored permanently  
Name - RecipeVault 🌟

## Features

* Add new recipes easily through a simple form
* Edit or update any recipe instantly
* Delete recipes from the database
* View all recipes in a clean layout
* Real time data storage and update using SQL

## Tech Stack

* HTML
* CSS
* JavaScript
* NodeJS
* ExpressJS
* EJS
* MySQL

## What I Learned

This project helped me understand how backend and database work together It taught me how to build and connect routes handle form data and make the frontend interact with a real database

## How to Run

1. Clone the repository
2. Install dependencies using `npm install`
3. Start MySQL and create a database named `food_manager`
4. Update the connection details in your app
5. Run the server with `node app.js`
6. Open your browser and visit `http://localhost:3000`

Simple command setup clean interface and fully functional CRUD operations make Food Manager a great beginner friendly full stack project
